## 📱 About

<blockquote>
<p><strong>FlowTube</strong> is an Android video player application that provides a YouTube-similar experience. Built as an alternative video streaming solution, it aims to deliver a smooth and intuitive video watching experience.</p>

### 🚀 Clone Instructions

<details>
<summary><strong>📥 Step-by-Step Cloning Guide</strong></summary>

#### 1. Clone the Repository
\`\`\`bash
git clone https://github.com/nahidislam/FlowTube.git
cd FlowTube
\`\`\`

#### 2. Open in Android Studio
- Launch Android Studio
- Select "Open an existing Android Studio project"
- Navigate to the cloned FlowTube directory
- Click "OK" to open the project

#### 3. Sync Dependencies
\`\`\`bash
# Android Studio will automatically prompt to sync
# Or manually sync using:
./gradlew build
\`\`\`

#### 4. Run the Project
- Connect your Android device or start an emulator
- Click the "Run" button (▶️) in Android Studio
- Select your target device

</details>
